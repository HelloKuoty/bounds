# 迭代纪要 · iter-50（执行轮 · 整局更多 procedural:分叉随 run 洗牌)

## 起点
- 基线绿(tests 130 / smoke OK)。承接 iter-48 复测 K哥/小鹿:"试炼从孤岛长成系统了、却焊在同一条直肠子上——前 7 关教学+传令所+Boss 每局走同一条线,变的是试炼内部、非整局形状;二周目'整局都新'还没看到。"

## 做了什么
- ✅ **分叉层按 run 种子洗牌发牌**:6 个手工分叉关(喧市/两面镇/连环渡/静室/契卷堂/名障镇)收进 `BRANCH_POOL`;`generate_map` 用 `run_seed` 洗牌后发到两个分叉层(各 3)。**每次重开,分叉选项的排布不同**(这层面对的三选一换了),同种子可复现(retry 公平)。
- ✅ **守住所有边界与不变式**:教学骨干(前 7 关 + 传令所)+ 试炼 + Boss **固定不动**(教学顺序不能随机);bazaar/sanctum 类型恒在池中(测试 `all_node_types_present` 仍过);每分叉层 3 个互异关(`branch_choices_differentiated` 仍过);所有分叉关都只用骨干已教的动词、均可解(smoke 随机分叉仍通关)。
- ✅ +测试 `test_branch_layout_varies_with_the_run_seed_but_reproduces`(跨 8 种子≥2 种分叉布局 + 同种子复现)。→ **131 绿**;smoke 通关;LOAD 0;重导出网页。

## 为什么这是对的(诚实边界)
- 整局可变的部分 = **非教学段(分叉选项排布 + 试炼内部结构)**,现在**都随 run 变**了:iter-44 让试炼内部每局不同,iter-50 让分叉选项每局不同 —— 合起来,"非教学部分的整局形状"随 run 变。
- **教学骨干必须固定**(lite 教学解谜的本质:动词要按依赖顺序教,不能随机)——所以"整局每一关都新"对本作不成立,这是**设计边界、非偷懒**;K哥的"深 roguelike 级整局随机"本就越北极星(见 iter-43/48)。本轮把"能随机的都随机了"。
