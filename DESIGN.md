# 界 / Bounds — 设计文档 v0.1

> 一个**寓言式**单人策略解谜 Roguelike。玩家在游戏里**从不听到一个软件术语**,
> 却在为了取胜的整理过程中,亲手把 DDD 的核心思想走了一遍。
> **设计文档里的 DDD 映射只对建造者可见(见附录 A);玩家面 100% 是寓言。**

---

## 0. 立意:寓言,而非术语

前作《领域之殇》的做法是"大声念出每个术语"——卡牌直接叫"聚合根""防腐层",但机制常是套皮。
本作反其道而行:

> **不要把 DDD 贴上去,要让 DDD 成为世界的物理定律。**
> 当游戏的物理是"边界与完整性 对抗 熵增",**好的玩法长得就和好的建模一模一样**——
> 不是我们强行教,而是玩家为了赢必须发现:
> *"必须一起变的东西要圈起来设一个守门人;同一个词在不同地方可以有不同意思,只要划清界、在边界翻译;放任不管,一切会糊成一团烂泥。"*
> 这几句话**就是 DDD 的灵魂**,玩家却以为自己只是在整理一片土地。

## 1. 中心寓言

> 你是一名**司守 (Steward)**,走入一片正在腐烂的国土。这里万物有"意义"——
> 意义清晰、归属分明、边界干净时,土地焕发**秩序 (Concord)**;
> 一旦意义含混、外人乱伸手、守门人倒下,**陈腐 (Rot)** 就蔓延,把一切吞成没有形状的沼泽。
> **你不靠打败陈腐取胜,你靠比它更会"组织"取胜。**

- **类型**:单人 Roguelike + 空间策略解谜(参考 Slay the Spire 的 run 结构 + 抽象整理解谜)
- **核心体验**:在限时(陈腐时钟)内,把一片混乱的土地切割、归束、翻译成有秩序的样子
- **反派**:不是怪兽,是**形状的消失**(熵)。终局 Boss 就是一片大泥球(**蔓沼**)。
- **一句话**:"一场对抗熵增的整理游戏——你画的每一条界,都在跟'万物糊成一团'的天性赛跑。"

### 基调
- 安静、肃穆、略带神话感。像在修复一幅褪色的古地图。
- 文字极简、留白多。村民/土地说的是寓言语言,绝不出现技术黑话。

## 2. 核心循环

### 一关之内(一个 territory)
```
土地降临(一片混乱散件) 
  → 观察:哪些名在打架?哪些东西必须一起变?哪些发生想被听见?
  → 整理:画界 / 成束设守门人 / 立真名 / 放译者石 / 升传令
  → 每回合:陈腐按"未解决的失稳数"上涨(时钟在走)
  → 秩序槽满 → 净化此地,前进;陈腐槽满 → 此地塌陷,run 受创/结束
```

### 一局之内(一次 run = 修复一个行省)
```
进入行省地图(~10–14 个节点,分叉) 
  → [土地 / 静室 / 集市 / 精英土地] × N
  → 抵达腹地(行省核心)→ 蔓沼 Boss
  → 净化腹地 = 通关 / 失败 = run 结束
```

## 3. 棋盘物理(核心机制,带数值)

> 这一节是游戏的"自然法则"。玩家通过试错和反馈**体会**它们,而非被告知。

### 3.1 棋子:生灵 与 筹码
- **生灵 (living)**:有持续身份,会记忆、结疤、成长。换掉一个生灵 = 丢失它的历史与增益。
- **筹码 (token)**:可互换。两枚同名同意的筹码毫无区别,可自由替换、堆叠。
- *(建造者注:生灵=Entity,筹码=Value Object。玩家只感到"这个我得认得是哪一个 / 这个无所谓哪一枚"。)*

### 3.2 名 与 真意(失稳的根源)
- 每个棋子带一个**名(glyph,可见符号)** 和一个**真意(meaning,它到底是什么)**。
- **规则**:在**同一片区域**内,一个**名**若对应**两种不同真意**,即产生一处**失稳 (instability)**。
- 失稳处会脉动闪烁(视觉),并**每回合播下陈腐**。
- 同名**同意**的多个棋子 → 不失稳(鼓励:一致的共同语言是好的)。
- *(建造者注:这是通用语言 + 限界上下文的核心张力。)*

### 3.3 界(画界 / wall)——头号动词
- **画界**:把一片区域里的若干棋子圈进一片**新区域**。
- **关键定律**:**同一个名在不同区域可以指不同真意,且完全合法**。
  所以"把两种打架的真意分到两片区域" → 失稳消除 → 即时得**秩序**。
- 这是玩家第一个、也是最深的"啊哈":**不要追求全局统一,要划界、就地定义。**
- *(建造者注:Bounded Context。这条定律是整个游戏的脊梁。)*

### 3.4 束 与 守门人(成束 / bundle + guardian)
- 有些棋子**必须保持彼此一致**(例:一本账册和它的每一条记账,总和必须对得上)。
- **成束**:把这样一簇圈起来,并指派其中一个为**守门人**。
- **定律**:外部效果(陈腐触手、别区的传令)想改动束内成员,**必须经过守门人**;
  绕过去直接戳 → 束的一致性破裂 → 散落 + 播陈腐。
- 守门人若倒下 → 整束散架(成员受创,筹码散落)。
- *(建造者注:Aggregate + Aggregate Root + 一致性边界。玩家学到"圈住要一起变的,所有访问走一个门"。)*

### 3.5 传令(升传令 / herald)
- 当一件重要的事发生(守门人倒下、一束完成、一个真名被立),可把它升为**传令**:一道别人能监听、并连锁反应的涟漪。
- 传令可链式触发(A 引发 B 引发 C),链越长收益越高,但每跳消耗一点**心力 (Insight)**。
- 跨区的传令必须经**译者石**中继(否则生接 = 播陈腐)。
- *(建造者注:领域事件 + Saga 编排 + 事件驱动集成。)*

### 3.6 译者石(放译者石 / translator)
- 两个真意体系不同的区域要互动时,边界上放一块**译者石**,把一边的意义翻成另一边的,接触才安全。
- 没有译者石的跨区生接 → 双方失稳 + 播陈腐。
- *(建造者注:防腐层 / 上下文映射。玩家学到"别让外来意义生吞进来,在边界翻译"。)*

### 3.7 陈腐 与 秩序(双时钟)
- **陈腐 (Rot / Blight)**:每回合 `+= 未解决失稳数 × 3`。还会沿"不清的界"和"超载的名"蔓延到邻近棋子。满 `blight_max` → 此地塌陷。
- **秩序 (Concord)**:每**解决一处失稳**(画对界、补上译者石、修好一束)`+= 1`。满 `concord_target` → 净化此地。
- 二者赛跑 = 核心张力。**乱点必输**:错误的整理会主动加速陈腐。
- *(建造者注:技术债复利 vs 整洁度。这条天然解决了前作"无脑点就赢"的问题——决策深度是物理,不是补丁。)*

### 3.8 腹地 与 心力(精力分配)
- **心力 (Insight)**:稀缺资源,用于强力动作(成束、长传令链、重塑)。每关有限,跨关可在静室补。
- **腹地 (Heartland)**:行省里唯一"你为之而来"的核心地,把心力堆在腹地收益最高;平摊到边角地会饿死核心。
- *(建造者注:核心域 + 精炼/蒸馏。)*

### 3.9 重塑(reshape / 重构)
- 可重画界、重组束、改派守门人——但改一片**已安定**的活区比塑造新区**更贵**(耗更多心力),且带着陈腐重塑很危险。
- *(建造者注:持续重构,越早越便宜。)*

## 4. 一关样例:渡口集市 (The Crossing)

> v1 第一关 = 最纯粹的"画界"教学。一片乱场漂着六个散件:

| 棋子 | 名 | 真意 | 类型 |
|---|---|---|---|
| 商人 | 商 | 商人 | 生灵 |
| 商人的账册 | **账** | 欠款 | 生灵 |
| 信使 | 信 | 信使 | 生灵 |
| 信使的清单 | **账** | 投递记录 | 生灵 |
| 钱币 | 钱 | 价值 | 筹码 |
| 钱币 | 钱 | 价值 | 筹码 |

- **冲突**:账册与清单都印着「账」,却一个意"欠款"、一个意"投递"。同处一片旷野 → 一处失稳,每回合播陈腐。
- **两枚钱币**:同名同意 → 不失稳(演示:可互换的筹码堆在一起没问题)。
- **解法**:画一道界,把信使 + 清单圈进一片新区域。此后「账」在两区各表一意 → 失稳消除 → 秩序 +1 → 净化。
- **时钟**:`blight_max=30`,每回合 +3,玩家约 10 回合内必须画对界。
- **玩家学到的(他不知道自己学到了)**:同名不同意要靠**边界**化解,而非强行统一。

> 后续关卡逐步引入:成束/守门人(账册 + 多条记账必须一致)、传令(投递完成 → 跨区销账,需译者石)、共享依赖精英关(两区抢一口井)。

## 5. Run 结构(复用前作底盘思路)

- 一局 = 修复一个**行省**:~10–14 个节点的分叉地图(复用 RunManager 地图生成 / 节点状态 / 存档模式)。
- 节点类型:
  - **土地**:一关整理解谜(普通)
  - **精英土地**:更毒的陈腐结构(例:**共井**——两区共享一口井,拆不开就两边渗陈腐)
  - **静室(休息)**:补心力,或深化一条已掌握的法则
  - **集市(商店)**:获取/升级**整理工具**(寓言道具,非术语卡)
  - **腹地 + 蔓沼(Boss)**:已塌一半、意义互渗的大泥球,限时重新切割回有序

## 6. 玩家的"牌库" = 整理工具(动词,不是术语卡)

一局内积累、升级、组合的工具(全寓言命名):

| 工具 | 作用 | (建造者:对应) |
|---|---|---|
| 界凿 | 画一道界(分区) | Bounded Context |
| 守印 | 成束 + 指派守门人 | Aggregate Root |
| 真名牌 | 在一区内为一个名固定真意 | Ubiquitous Language |
| 译者石 | 边界翻译 | ACL / Context Map |
| 传令号角 | 把一个发生升为传令 | Domain Event |
| 断线刃 | 切断脆弱的"提线"(强迫行为本地化) | 充血模型/去贫血 |
| 腹地之眼 | 标记腹地,心力投入翻倍 | Core Domain 精炼 |

> 升级 = 改变效果而非单纯加数值(参考 Slay the Spire 更有趣):例如界凿升级后可"一次画两道界"。

## 7. 教程哲学:只用寓言教

- 没有 tooltip 写"这是限界上下文"。第一关用**纯视觉**教:失稳处闪红 + 陈腐逼近 → 玩家试着画界 → 红光熄灭 + 秩序亮起。**反馈即教学。**
- 一句旁白足矣:"两个世界共用一个词,土地便不安。"(绝不说"这就是边界划分")。

## 8. 美术 / 音频基调(占位即可,不阻塞)

- **美术**:极简几何 + 古地图羊皮纸质感。生灵=带轮廓的图腾,筹码=小圆币,界=发光的线,陈腐=蔓延的墨渍,秩序=暖金光晕。占位用程序生成色块/线条即可,缺图不崩。
- **音频**:环境 drone + 画界时的"咔哒"确定音 + 陈腐蔓延的低频不安 + 净化时的舒展和弦。
- 详见 [art/ART_BIBLE.md](art/ART_BIBLE.md)(待写)。

## 附录 A:物理定律 ↔ DDD 映射(**仅建造者可见,严禁泄露到玩家面**)

| 棋盘物理(玩家面) | DDD 思想(禁止出现) | 玩家被迫养成的手感 |
|---|---|---|
| 生灵 / 筹码 | 实体 / 值对象 | 区分"在乎是哪一个" vs "只在乎多少" |
| 名 / 真意 / 失稳 | 通用语言 | 同一语境内,词必须精确唯一 |
| 画界 | 限界上下文 | 划界、就地定义,别追求全局统一 |
| 束 / 守门人 | 聚合 / 聚合根 | 圈住要一起变的,访问走一个门 |
| 守门人倒 → 束散 | 一致性边界崩溃 | 根没了,整簇失去一致性 |
| 传令 / 涟漪链 | 领域事件 / Saga | 让重要发生成头等公民,以涟漪联动 |
| 译者石 | 防腐层 / 上下文映射 | 别让外来意义生吞,在边界翻译 |
| 腹地 / 心力 | 核心域 / 精炼 | 把最好功夫下在差异化处 |
| 陈腐 / 蔓沼 | 技术债 / 大泥球 | 债复利,草率边界会烂成一团 |
| 重塑 | 重构 | 趁早重构,越拖越贵 |
